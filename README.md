# Hospital Command Center Dashboard

## Project Overview

The Hospital Command Center Dashboard is a healthcare analytics project developed using Streamlit and Python. The dashboard helps monitor hospital operations including patient journey, ICU occupancy, medical billing, emergency room activity, staff allocation, and operation theatre management using synthetic healthcare datasets.

## Features

- Interactive healthcare dashboard
- Multiple hospital datasets
- Filters and search functionality
- Bar charts and pie charts
- Machine learning prediction models
- Dark theme dashboard
- Streamlit Cloud deployment

## Datasets Used

1. Patient Journey Dataset
2. Bed Occupancy / ICU Dataset
3. Medical Billing Dataset
4. Emergency Room Dataset
5. Staff Allocation Dataset
6. Operation Theatre Dataset


## Technologies Used

- Python 3.10+
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- Faker
- Git & GitHub
- Streamlit Cloud
